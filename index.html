Bio

·

I'm Runjie Guan, a second-year student in UC San Diego. People nowadays don't like read long paragraphs, so I'll keep my introduction as short as possible.

Online social media has shaped us in a way that we want to grab as much as possible information in a period of time as short as possible. We read, but shallowly. We write, but shortly. We listen, but unconsciosly. We think, but superficially.

But that's not the reason I love simplicity. It's a principle that I stick to promote efficiency and clarity. In Computer Science, they call it performance.


My first program was written about seven years ago, in my middle school. But I was not interested in this then. Several years later in 2011, as a high-school student I started to learn Java using a book in which most of the examples wouldn't even compile. I was a kiddo then, knowing nothing about the importance of a good textbook, but I still managed to finish my first Java Applet, by which I helped myself memorizing English words.

Unfortunately I lost its source code, which had anything that I needed to learn in my first programming course in my college.

After that my journey with code began. I love to code out of pure interest, and thus love everything programmable and customizable. Because of that, I write code to make myself easier and happier, and hopefully benefit others.


Besides that I also play other stuffs. I play Rubik's cube (personal best 21.39', speed cubing), billiards. I play Minecraft (because it is highly programmable and creative) and FPS video games like Call of Duty. I play Ingress, as the second player to reach highest level here in San Diego.

JournoxicHTML/CSS/JavaScript, 30K lines

An online journal keeper with OneDrive

Apr 2015 - Now

People don't write stuffs these days, but I keep what is going on in my life weekly, if not daily.

Demand determines supply and quality - this is how I understand today's Internet market. Among all the friends I've met in my life, nobody regularly writes down what they have experienced, not to mention the quantity of good apps that help you do that. There are, of course, those with either limited features or limited platform support.

Then I decide to create one on my own, whose core part took me three months to write, with 30K lines of HTML/CSS/JavaScript. This webapp will store the data on user's OneDrive account, and can read uploaded pictures and videos from their phones.

Since there seems to be only me using this stuff, I've made it to be highly personalized for my own life pattern. And because OneDrive does not include CORS support for code flow in its authenticating steps, I have to install a Chrome extension to fix this problem.

More info about this website can be found on the GitHub page.

Sep 2015

After installing this small script, a link will be created on the Webreg page. Then you can download a file which later can be imported to Google Calendar.
